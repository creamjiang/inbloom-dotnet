Setting Up the .NET Library

Begin setup by downloading the following:

  • The .NET Library solution, containing the ApiClassLibrary itself and sample application code, available from https://github.com/upeo/inbloom-dotnet

  • The .NET Library reference, which is an index and supporting files with documentation about each class and method in the library, available from this link: https://github.com/upeo/inbloom-dotnet

.NET Library: Applications  Managed with IIS

The easiest way to get started with the library is to use IIS (Internet Information Services) to manage the dependencies. If you're not using IIS, though, you can use the development web server built into Microsoft Visual Studio. The table included in the steps below lists the required dependencies and where to obtain them.

  1. Unzip the dotnet library code.

  2. Configure this application in IIS. This setting must match the redirect URL configured in your inBloom sandbox when the application was registered.

  3. Open it in Visual Studio and run the application.
